## 📌 Overview

PANOR is a government-grade, AI-powered healthcare infrastructure platform designed to unify Pakistan’s medical ecosystem into one secure, intelligent system.

It connects:
- Patients  
- Doctors  
- Laboratories  
- Financial systems  
- Government health authorities  

All powered by real-time AI-driven clinical intelligence.

## 🚀 Getting Started

### Clone the Repository
```bash
git clone https://github.com/your-username/panor.git
cd panor
```

### Install Dependencies
```bash
npm install
# or
pip install -r requirements.txt
```

### Run Locally
```bash
npm start
# or
python app.py
```
